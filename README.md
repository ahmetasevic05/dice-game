# dice-game
Dice game including two players.First player to 100 points wins!

The goal here is to roll the dice as long as you don't get number 1 on the dice.In that case it is opponents turn.Your score adds up as long as you don't get one,in that case the current score deletes.You also have the option to hold the current score,wich toggles the turn to the opponent.The first one to get 100 points or above,wins.
